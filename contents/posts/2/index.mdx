`styled-component`는 `react` 생태계에서 가장 많이 사용되는 style 라이브러리 중 하나입니다.
자바스크립트 파일 내에서 CSS를 사용할 수 있게 해주는 대표적인 CSS-in-JS 라이브러리이며 `SASS`처럼 스타일 코드의 재사용도 가능합니다.

아래는 `styled-component` 를 사용한 스타일 재사용 예제 코드입니다.

```ts:src/styles/mixin.ts
import { css } from 'styled-components';

export const reset = css`
  box-sizing: border-box;
  margin: 0;
  padding: 0;
  font-family: inherit;
  font-weight: inherit;
  font-size: inherit;
  line-height: inherit;
  color: inherit;
  vertical-align: top;
  text-underline-position: under;
  background-repeat: no-repeat;
  mask-repeat: no-repeat;
`;

export const ellipsis = (line: number) => css`
  overflow: hidden;
  text-overflow: ellipsis;
  display: -webkit-box;
  -webkit-box-orient: vertical;
  -webkit-line-clamp: ${line};
`;

export const device = {
  mobile: `(max-width: 768px)`,
};
```

사용 방법은 다음과 같습니다.

```ts:src/styles/reset.ts
import { createGlobalStyle } from 'styled-components';
import { reset } from '@/styles/mixin';

export const GlobalStyle = createGlobalStyle`
  *,
  :before,
  :after {
    ${reset};
  }
  .ellipsis {
    ${ellipsis(2)};
  }
  @media ${device.mobile} {
    min-width: 320px;
  }
`;
```
