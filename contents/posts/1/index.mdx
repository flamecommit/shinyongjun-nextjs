---
title: 블로그 리뉴얼 (by Next.js 13)
date: 2023-08-11T09:00:00
categories: ['next.js']
---

`vue.js` 를 사용하던 회사에서 `react` 로의 스택변화가 있어 공부 목적으로 블로그를 리뉴얼 하였습니다.
리뉴얼은 `next.js` 라는 react 기반의 SSR 프레임워크를 사용했으며 제일 최신버전인 13버전을 채택했습니다.

`next.js` 프레임워크는 SSR 빌드와 SSG 빌드를 모두 제공하는데, 이 블로그처럼 github pages를 사용하기 위해선 SSG 방식의 빌드 결과물이 필요합니다.

블로그 포스팅은 `mdx` 파일확장을 사용했습니다. `mdx`란 마크다운 파일에서 직접 `jsx`를 작성할 수 있는 마크다운의 상위 집합입니다.
`mdx` 파일 렌더링은 `react-mdx-remote` 모듈을 사용했습니다.

`nuxt.js` 를 사용하여 블로그를 만들땐 이미 지어진 공장에서 제품만 찍어낸다는 느낌이었는데
`next.js` 는 공장 설비부터 직접 개발하는 느낌이 들어 좀더 재미를 느낄수 있었습니다.

앞으로 이 블로그를 통하여 블로그에 적용한 기술이나 공부한 내용들을 포스팅 할 계획입니다.
